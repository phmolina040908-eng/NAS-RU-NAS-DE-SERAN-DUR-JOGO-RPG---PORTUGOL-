# NAS-RU-NAS-DE-SERAN-DUR-JOGO-RPG---PORTUGOL-
Meu primeiro projeto voltado à programação estruturada no Portugol, realizado no decorrer do mês de Setembro ao mês de Novembro de 2025 no início do curso de Desenvolvimento de Sistemas. Nesse projeto, apreendi habilidades e competências voltadas à lógica de programação, programação estruturada e exercício de criatividade.
